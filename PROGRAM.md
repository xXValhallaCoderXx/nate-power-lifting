## 2. Hard constraints (do not violate)

1. **Back is coming off injury.** Never let pulling volume days become grinders. Most
   deadlift volume stays submaximal (≤75% TM). Only ONE pull session per week is heavy.
2. **Legs are reserved for running.** Squat only **1×/week**, treated as
   maintenance + light growth — never the priority, never a leg-destroying session.
3. **Masters lifter (40).** Thorough warm-ups (especially bench shoulder/elbow), more
   frequency at lower per-session intensity, generous recovery. Recovery — not effort or
   motivation — is the limiter.
4. **Autoregulate.** RPE governs over prescribed kg. A bad day → go lighter, no guilt.
5. **Don't chase old maxes.** The 137.5 bench / 250 pull are learned and will return;
   rushing the Training Max is how the injury returns.

## 3. Training philosophy / theory

The program is built on the **Greg Nuckols / Stronger by Science high-frequency model**,
because it fits this athlete's needs precisely:

- **High frequency, moderate intensity.** Hitting each lift 2–3×/week gives lots of skill
  practice and accumulated volume without any single session being brutal — ideal for
  rebuilding muscle memory AND protecting a recovering back.
- **Heavy / Medium / Light (HML) daily undulation.** Each lift gets three *jobs* across the
  week (heavy top set, volume, speed/technique) rather than the same stimulus repeated.
- **Submaximal bias.** ~90% of work sits in the 60–80% range, consistent with both SBS
  programming and return-from-layoff best practice.
- **AMRAP-driven autoregulation.** A top "as-many-reps-as-possible" set periodically
  recalibrates the Training Max, so progression is earned by performance, not a fixed
  spreadsheet curve.

**Return-from-layoff principles applied** (from JTS, SBS, PowerliftingTechnique consensus):
- Base loads on ~85–90% of estimated capability, then start with roughly **half** the volume
  you think you can handle and build up.
- Ramp intensity over 6+ weeks (8–12 for true heavy loads).
- Use 2–3×/week frequency deliberately — more practice reps at lighter loads rebuild tissue
  tolerance faster than heavy singles.

**Key reference resources:**
- Nuckols "28 Programs" — modular weekly templates (squat/bench/deadlift × beginner/
  intermediate/advanced × 1×/2×/3× weekly), 4-week cycles, retest after each.
  Free via Stronger by Science newsletter signup.
- SBS "Returning to Training" article; JTS "Rules for the Deconditioned Lifter."

## 5. Weekly structure

**4 lifting days/week.** Bench 3×, Deadlift 3×, Squat 1×. Runs go on off-days or *after*
lifting; keep the hardest run **away** from heavy squat (Day 2) and heavy deadlift (Day 3).

| Day | Main lifts & roles |
|-----|--------------------|
| **Day 1** | Bench = **HEAVY** (top set + back-offs) · Deadlift = **MEDIUM/volume** |
| **Day 2** | Squat = main (only squat day) · Bench = **MEDIUM/volume** |
| **Day 3** | Deadlift = **HEAVY** (top set + back-offs) · Bench = **LIGHT/speed** |
| **Day 4** | Deadlift = **LIGHT/speed or variation** · Accessory Bench (3rd session) |

This yields Bench 3× (heavy/medium/light), Deadlift 3× (medium/heavy/light), Squat 1×.

## 6. Session detail (per main lift)

Percentages are off Training Max (§4). All main lifts: ramp up in small jumps to warm up.

- **HEAVY day:** ramp to one top set of **3–5 reps @ ~80–85% TM**, taken as a hard-but-clean
  AMRAP (early in a block leave 1–2 reps in reserve). Then **2–3 back-off sets at –12%** for
  the same reps.
- **MEDIUM/volume day:** **4–5 sets × 5–6 reps @ 70–75% TM.** Bread-and-butter volume.
- **LIGHT/speed day:** **6–8 sets × 2–3 reps @ 65–70% TM**, every rep explosive, perfect
  form. For deadlift, use a **variation** here (deficit / paused / RDL / tempo).

Rest: ~3 min on main work, ~90 s on accessories.

### Accessories (≈4/day, 2–3 sets, RPE 7–8 — recovery-limited, keep honest)

Leg Curl *(not RDL — spare runners' hamstrings)* 3×12 · Cable Triceps 3×15 · Ab Wheel/Plank 3×.

**Day 3 (DL heavy + speed Bench):** Barbell/Pendlay Row 3×8 · Back Extension (light) 3×12 ·

## 7. Progression scheme

Two phases, run in sequence.

### Phase 0 — Reintroduction (Weeks 1–3, linear)
Strength returns fast via muscle memory, so a simple linear ramp is most efficient while
connective tissue catches up.
- Deliberately easy TM. Add **~2.5 kg to bench TM**, **~5 kg to squat/DL TM** each week.
- **No AMRAPs yet** — stop top sets ~2 reps shy. Goal = practice + tissue tolerance, not PRs.

### Phase 1+ — Nuckols-style waves (4-week cycles, undulating)
Begin once movement feels normal (~week 4).

| Week | Heavy-day top set |
|------|-------------------|
| 1 | ~80% TM, **5+ rep AMRAP** |
| 2 | ~82.5% TM, **4+ rep AMRAP** |
| 3 | ~85% TM, **3+ rep AMRAP** |
| 4 | ~65% TM, low volume — **deload** |

**Autoregulation rule (the core mechanic):** on each heavy-day AMRAP —
- Beat the rep target by **2+** → bump TM next cycle (**+2.5 kg bench, +5 kg squat/DL**).
- Just hit it → small bump.
- Miss → keep TM the same, run the cycle back.

Retest a true single only every 2–3 cycles, or never — let rep PRs tell the story.

**Why waves long-term:** at this training age linear stalls within weeks; the wave varies
stress, lets him express strength on the AMRAP while training mostly submaximal, and is
forgiving of the off-days that come with age + running.

## 8. Setting starting weights (method, for any new lift)
Find a weight that is a clean, controlled **5 reps @ RPE 7** (3 in reserve). That ≈ 75% of a
working 1RM. Then **TM = (that weight ÷ 0.75) − 10%** during the return phase.

## 10. Glossary
- **TM (Training Max):** the number all % are based on; ≈ a conservative current 1RM.
- **AMRAP:** As Many Reps As Possible — a top set taken near failure to gauge/drive progress.
- **RPE:** Rate of Perceived Exertion, 1–10. RPE 7 ≈ 3 reps left; RPE 8 ≈ 2 left; RPE 9 ≈ 1.
- **HML:** Heavy / Medium / Light daily undulation.
- **Back-off sets:** sets at a reduced load (here −12%) after a heavier top set.
